Anatomy of a Skill

Every skill follows the same shape.

The folder

my-skill/
├── SKILL.md Required. Frontmatter + instructions.
├── scripts/ Optional. Code Claude can run.
├── references/ Optional. Docs loaded only when needed.
└── assets/ Optional. Templates, fonts, icons used in output.

The frontmatter sits at the top of SKILL.md

--- name: status-report-builderrequired description: Generate a client status report using the STATUS framework.required Use whenever an FDE needs a weekly update, a delivery summary, or a red/amber/green health check for an engagement. ---
Below the frontmatter, the instructions. Above it, the two fields that make the skill discoverable.
