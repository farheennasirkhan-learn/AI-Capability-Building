The SKILL Framework

Five principles behind every skill worth keeping. The word is the reminder.

S K I L L
S
Scope it One job, clearly bounded. Narrow skills are reliable skills.
K
Keep it lean Load only what is needed, when it is needed.
I
Instruct with intent Explain the why, not a wall of rigid rules.
L
Lead with the trigger The description decides whether the skill ever fires.
L
Loop to improve Draft, test, review, refine. Skills are iterated, not authored.
A good skill is scoped, lean, reasoned, discoverable, and tested.
